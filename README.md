# Mi proyecto de Solana

Este proyecto es un ejemplo de un programa en Solana creado usando Anchor.

## Funciones

- initialize(): crea una cuenta contador
- incrementar(): aumenta el valor del contador

## Tecnologías utilizadas

- Solana
- Anchor
- Rust
- TypeScript

## Estructura del proyecto

client/ → código del cliente  
src/ → programa principal en Rust  
tests/ → pruebas del programa
